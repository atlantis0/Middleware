    An Overlay Network Middleware on Mobile Device Ensemble

    Project maintained by Sirack Hosted on GitHub Pages — Theme by mattgraham

    Introduction

    Middleware is a layer that exists between any two components. It acts as a mediator for the components and provides an extra capability for flexible communication. Introducing such layer has is own trade-off but the advantage weigh more than the overhead it introduces.

    Aim

    Is to design a platform independent Middleware that facilitates wireless communication on mobile devices by using simple interfaces available through the library designed here.

    $ git clone https://github.com/Sirack/Middleware.git
$ git checkout with_tcp

    Design

    The design of the Middleware is divided into three parts. The first section is network creation where devices send their connection profile to the central access point.
